Welcome to Auto Mail's documentation!
=====================================

API Reference
-------------

Automail is a `Python` package that provides a simple interface to send emails.
It is built on top of Python's built-in smtplib and email modules.
It is designed to be simple to use and to handle failures gracefully.


.. note::

   Automail is currently in alpha. It is not yet ready for production use.

Please see the `tutorial` for a quick introduction on how to use Automail.
